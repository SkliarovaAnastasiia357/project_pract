# Teamnova Project Memory

Use this file as the index for durable project context.

## Read First

- `active.md` — current task, branch, assumptions, next steps.
- `handoff.md` — latest continuation brief for a fresh session.
- `commands.md` — verified build/test/run commands and local caveats.

## Topic Files

- `issues.md` — verified bugs, limitations and fixes.
- `progress.md` — durable milestone log.

## Notes

- Do not store secrets or raw chat transcripts here.
- Keep facts separate from verified fixes.
