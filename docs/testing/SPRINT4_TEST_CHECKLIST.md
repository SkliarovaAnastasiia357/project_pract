## Заявки

- [ ] Отправка заявки
  Шаги: участник находит чужой проект и нажимает "Откликнуться".
  Ожидаемый результат: показывается успешное сообщение, карточка проекта получает статус заявки.

- [ ] Запрет повторной заявки
  Шаги: повторно отправить заявку в тот же проект.
  Ожидаемый результат: API возвращает конфликт, новая заявка не создается.

- [ ] Просмотр входящих заявок
  Шаги: владелец открывает `/requests`.
  Ожидаемый результат: видит заявку, проект, кандидата, его bio и навыки.

- [ ] Принятие заявки
  Шаги: нажать "Принять".
  Ожидаемый результат: статус заявки меняется на "Принята".

- [ ] Отклонение заявки
  Шаги: нажать "Отклонить".
  Ожидаемый результат: статус заявки меняется на "Отклонена".
