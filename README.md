## Преимущества выбранного тех. стека

### React
Обеспечивает создание динамичного и отзывчивого интерфейса. Позволяет удобно масштабировать проект за счёт компонентного подхода.

### TypeScript (на фронте)
Добавляет строгую типизацию, снижает количество ошибок и упрощает поддержку кода в долгосрочной перспективе.

### CSS
Гибко управляет стилями и адаптивностью интерфейса, позволяет создавать удобный и современный UI.

### Golang (backend)
Высокая производительность и эффективная работа с конкурентностью (goroutines). Отлично подходит для масштабируемых веб-сервисов.

### PostgreSQL
Надёжная реляционная база данных с поддержкой сложных запросов, высокой целостностью данных и хорошей производительностью.
